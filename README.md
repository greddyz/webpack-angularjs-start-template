# Webpack AngularJS start template

## Install
```sh
$ npm install
```
### Config and Settings
/webpack.config.js

### Dev
```sh
$ npm run dev
```

### Dev with watch to changes in files
```sh
$ npm run watch
```
In your browser, navigate to: http://localhost:8080/


### Production
```sh
npm run build  
```
Copy everything in build/ folder to the server.
